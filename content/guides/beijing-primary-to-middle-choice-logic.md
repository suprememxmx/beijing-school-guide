小升初选学校和幼升小的逻辑完全不同。幼升小主要看"家门口有什么"，小升初是"能去哪里就尽量去最好的"。

## 一、选初中的核心指标

### 1. 中考成绩

中考成绩是衡量初中质量的核心指标。重点看：
- 高分段比例（能上优质高中的比例）
- 平均分（学校的整体水平）
- 校额到校的名额分配

### 2. 学校管理风格

**严格管理型：** 师达中学等，学习氛围浓厚，适合自律性不够强的孩子。
**开放自主型：** 十一学校等，走班制，适合自觉性强的孩子。

### 3. 师资力量

- 高级教师比例
- 骨干教师数量
- 教师的稳定性

### 4. 地理位置

初中三年，每天通勤时间超过40分钟会比较辛苦。住宿是另外一个选择。

## 二、公办 vs 民办

| 维度 | 公办初中 | 民办初中 |
|------|---------|---------|
| 学费 | 免费 | 6-16万/年 |
| 入学方式 | 派位为主 | 自主招生 |
| 生源 | 派位，多样化 | 筛选，相对整齐 |
| 教学 | 统一标准 | 各校自有特色 |
| 中考成绩 | 头部公办强 | 部分民办很突出 |

**建议：** 如果派位结果理想（优质公办），没必要上民办。如果派位不理想，民办是很好的备选。

## 三、大校 vs 小校

**大学校：** 招生多，资源多，竞争激烈，但孩子容易被淹没。
**小学校：** 关注度高，氛围温馨，但资源可能有限。

## 四、校额到校的影响

校额到校政策让"选初中"多了一层考量：

- 在普通初中排名靠前 → 可以通过校额到校进优质高中
- 在优质初中排名靠后 → 可能不如在普通初中通过校额到校上好高中

这是一个需要权衡的问题。

## 五、决策框架

```
第一步：看派位结果 → 理想 → 接受
                    → 不理想 → 考虑民办

第二步：如果有多个选择 → 对比中考成绩
                        → 对比管理风格
                        → 对比距离

第三步：考虑长远 → 校额到校的影响
                  → 高中的升学路径
```
