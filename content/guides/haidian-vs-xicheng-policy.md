海淀和西城是北京教育的"巅峰对决"。两个区聚集了全市最多的优质教育资源，但入学政策的差异决定了不同家庭在两个区的入学结果可能天差地别。

## 一、入学方式对比

| 维度 | 海淀区 | 西城区 |
|------|-------|-------|
| 划片模式 | 多校划片为主 | 多校划片为主 |
| 基准日 | 1911（2019年1月1日） | 731（2020年7月31日） |
| 单校划片 | 1911前购房可单校 | 731前购房可单校 |
| 学位紧张度 | 极高 | 极高 |
| 落户年限要求 | 部分学校要求满1年 | 暂无统一要求 |

### 关键数字解读

**海淀1911政策**：2019年1月1日后在海淀区购房的家庭，全部通过多校划片方式入学。也就是说，2019年后买学区房，不再保证能上对应学校。

**西城731政策**：2020年7月31日后在西城区购房的家庭，全部通过多校划片方式入学。和海淀一样，买房不再等于锁定学位。

## 二、入学顺位对比

### 海淀区顺位（简化版）

```
第一顺位：父母房产 + 父母/孩子户籍均在房产上
第二顺位：父母房产 + 孩子户籍在房产上  
第三顺位：四老房产 + 父母无房
第四顺位：父母房产 + 户籍不在
第五顺位：非京籍有房
第六顺位：非京籍无房
```

### 西城区顺位（简化版）

西城区的顺位比海淀更细致，各校执行的差异也更大。总体趋势是：

- 西城对"户籍+房产"的一致性要求更高
- 西城的四老房产顺位靠后于海淀
- 西城对非京籍的要求是全北京最严的

## 三、多校划片执行力度

| | 海淀 | 西城 |
|--|------|------|
| 执行时间 | 2019年起 | 2020年起 |
| 严格程度 | 严格 | 非常严格 |
| 派位范围 | 学区内派位 | 学区内派位 |
| 调剂情况 | 部分调剂到相邻学区 | 调剂也有，但范围较小 |

## 四、教育资源对比

### 小学层面

| | 海淀 | 西城 |
|--|------|------|
| 顶尖小学数量 | 4-5所 | 3-4所 |
| 第二梯队数量 | 10+ | 8+ |
| 教育风格 | 偏竞争、多样化 | 偏均衡、传统 |
| 课外资源 | 极其丰富 | 丰富 |

### 小升初出口

**海淀**：海淀的小升初路径更加多元化——有"六小强"等优质初中，也有大量民办初中。优秀学生有更多突破派位限制的机会。

**西城**：西城的小升初以派位为主，对口直升比例高于海淀。进入优质初中的确定性更高，但"超常发挥"的机会更少。

## 五、房价与入学性价比

| | 海淀 | 西城 |
|--|------|------|
| 学区房均价 | 8-15万/㎡ | 9-16万/㎡ |
| 上车门槛 | 400万起 | 350万起 |
| 溢价程度 | 高 | 最高 |
| 多校后溢价变化 | 有所回落 | 有所回落但坚挺 |

## 六、怎么选？

### 选海淀

- 重视学术竞争，希望孩子在竞争中成长
- 孩子在某方面有特长（科技、艺术等）
- 家长愿意投入时间和精力参与教育
- 能接受入学的不确定性

### 选西城

- 追求教育均衡和确定性
- 希望降低"派位到差校"的风险
- 家长工作繁忙，希望学校承担更多教育责任
- 预算充足，能承受学区房溢价

## 常见问题

**Q：1911和731之后在海淀/西城买房还能上好学校吗？**
A：能，但不保证。多校划片只是降低了确定性，不是完全没机会。大部分孩子仍然能被派到片区内较好的学校。

**Q：海淀和西城哪个适合非京籍？**
A：都不太适合，但非要选的话，海淀对非京籍稍微友好一些。西城的非京籍社保要求高达一年，是全北京最高的。
