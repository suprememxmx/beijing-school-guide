## 一、中考科目与分值

北京中考的考试科目和分值：

| 科目 | 分值 | 考试形式 |
|------|------|---------|
| 语文 | 100分 | 笔试 |
| 数学 | 100分 | 笔试 |
| 英语 | 100分 | 笔试+听说机考（40+60） |
| 物理 | 80分 | 笔试+实验操作 |
| 道德与法治 | 80分 | 笔试（开卷） |
| 体育与健康 | 40-60分 | 现场考试 |
| **总分** | **500-520分** | |

## 二、各科复习建议

### 语文（100分）

**考试内容：** 基础知识、古诗文阅读、现代文阅读、作文。

**备考建议：**
- 古诗文默写是送分题，必须拿满
- 阅读理解靠练，每天一篇保持手感
- 作文提前准备几个素材模板（成长类、亲情类、社会类）

### 数学（100分）

**考试内容：** 代数、几何、统计与概率、综合题。

**备考建议：**
- 基础题（80%）不能丢分——计算要准、步骤要全
- 压轴题（20%）——平时多练，考试时先易后难
- 错题本是最好的复习资料

### 英语（100分）

**考试内容：** 听说机考+笔试。

**备考建议：**
- 听说机考可以提前练——官方的模拟系统一定要用
- 单词量是基础，考纲词汇必须全部掌握
- 阅读理解占分高，每天精读一篇

### 物理（80分）

**考试内容：** 力学、电学、热学、光学、实验操作。

**备考建议：**
- 力学和电学是重点，占60%以上
- 实验题考的是原理和分析，不是背步骤
- 公式要理解，不要死记硬背

### 道德与法治（80分，开卷）

**备考建议：**
- 虽然是开卷，但时间有限，不能每道题都翻书
- 熟悉教材目录，知道知识点在哪一页
- 结合时政热点答题

### 体育（40-60分）

**考试项目：** 长跑、球类、力量类等。

**备考建议：**
- 体育分是最好拿的——靠练就能提分
- 提前半年开始针对性训练
- 考试时注意技巧（如跑步的节奏分配）

## 三、复习时间线

| 时间 | 重点任务 |
|------|---------|
| 初三上学期 | 扎实学完新课，打好基础 |
| 寒假 | 第一轮系统复习，查漏补缺 |
| 3-4月 | 第二轮复习，专项突破 |
| 5月 | 模拟考试训练，适应考试节奏 |
| 6月 | 考前调整，心态管理 |
