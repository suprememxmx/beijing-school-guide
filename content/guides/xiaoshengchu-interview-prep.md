## 一、哪些学校需要面试

在小升初中，以下情况通常需要面试：
- 民办初中的自主招生
- 部分公办初中的寄宿班、特色班
- 特长生选拔

**不需要面试的情况：**
- 正常派位（不需要面试）
- 对口直升（不需要面试）

## 二、面试的核心考察内容

### 学术能力

**语数外三科的学业水平**是面试中最常考察的。

**语文：** 阅读理解、口头表达、古诗词积累
**数学：** 逻辑思维、应用题解题思路（部分学校有笔试环节）
**英语：** 简单对话、英语自我介绍、英语短文朗读

### 综合素质

- **逻辑思维**——遇到问题怎么思考和表达
- **语言表达**——清晰有逻辑地讲述一件事情
- **行为习惯**——面试中的礼仪和态度
- **特长展示**——艺术、体育、科技等方面的成果

### 行为观察

面试过程中，老师会观察孩子的：
- 是否自信、大方
- 是否有礼貌
- 是否善于沟通
- 是否对学习有兴趣

## 三、小升初简历怎么写

### 简历基本结构

```
1. 基本信息（姓名、学校、联系方式）
2. 学业成绩（主科成绩、获奖情况）
3. 综合素质（特长、社会实践、获奖记录）
4. 自我评价（孩子的自我介绍）
5. 家长寄语（可选）
```

### 注意事项

1. **简洁明了**——不要超过3页
2. **突出重点**——把最重要的奖项和经历放在前面
3. **真实准确**——不要夸大或虚构
4. **重视细节**——排版整齐、无错别字

## 四、面试准备的时间线

| 时间 | 准备事项 |
|------|---------|
| 五年级暑假 | 开始了解目标学校的面试要求 |
| 六年级上 | 准备简历初稿 |
| 12月-1月 | 参加目标学校的开放日，获取第一手信息 |
| 2月-3月 | 完善简历，开始面试模拟练习 |
| 4月-5月 | 正式面试，调整状态 |

## 五、给家长的建议

1. **不要过度准备**——面试主要看孩子的基本素质，不是比谁背得多
2. **孩子的状态最重要**——面试当天让休息好，保持自然状态
3. **家长不必太焦虑**——面试只是小升初的一部分，派位才是最主流的途径
4. **真实比"包装"重要**——面试老师见过很多"包装"过的孩子，真实的、有礼貌的孩子反而更受欢迎

## 六、常见问题

**Q：所有民办初中都有面试吗？**
A：大多数民办初中有面试环节，但面试形式和内容因学校而异。建议提前了解目标学校的具体要求。

**Q：面试被拒绝了还能参加派位吗？**
A：可以。民办校面试和派位是两条线，不影响派位资格。
