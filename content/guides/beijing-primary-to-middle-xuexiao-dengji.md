## 一、海淀区初中梯队

### 第一梯队

| 学校 | 特点 |
|------|------|
| 人大附中 | 全国顶尖，竞赛+高考双强 |
| 清华附中 | 理科强校，创新培养 |
| 十一学校 | 走班制，个性化培养 |
| 北大附中 | 素质教育，课程创新 |
| 101中学 | 文科传统，校园最美 |

### 第二梯队

| 学校 | 特点 |
|------|------|
| 首师大附中 | 教学严谨，升学稳定 |
| 理工附中 | 理科特色 |
| 交大附中 | 管理规范 |
| 八一学校 | 全面均衡 |
| 中关村中学 | 老牌优质 |

### 第三梯队

二十中、北航附中、海淀实验中学等

## 二、西城区初中梯队

### 第一梯队

北师大附属实验中学、北京四中、三帆中学、北京八中

### 第二梯队

北师大二附中、一六一中学、三十五中、十三中、十五中

### 第三梯队

铁二中、十四中、四十三中等

## 三、东城区初中梯队

### 第一梯队

五中分校、二中分校

### 第二梯队

一七一中学、汇文中学、东直门中学

### 第三梯队

二十二中、五十中、六十五中等

## 四、朝阳区初中梯队

### 第一梯队

八十中学、陈经纶中学

### 第二梯队

清华附中朝阳学校、人大附中朝阳学校、日坛中学

### 第三梯队

工大附中、东方德才学校等

## 五、选择建议

1. **第一梯队竞争激烈**——如果孩子成绩拔尖，可以冲刺
2. **第二梯队性价比高**——教学质量好、入学难度适中
3. **第三梯队的实验班值得关注**——部分学校的实验班教学质量不输第二梯队
4. **梯队只是参考**——具体选择还要结合孩子的实际情况
