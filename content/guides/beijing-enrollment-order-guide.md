很多家长以为"在划片范围内就能上对应学校"，这是个误区。北京小学录取是按**顺位**来的——顺位靠前的先录，录满为止。

## 一、什么是入学顺位？

入学顺位是指区教委根据家庭的户籍、房产等情况，对学生进行排序录取的规则。

**核心逻辑**：顺位越靠前，进入优质校的概率越大；顺位越靠后，越可能被调剂到有空余学位的学校。

## 二、通用顺位规则（以某区为例）

各区顺位大同小异，下面是典型顺位：

```
第一顺位：父母房产 + 父母/孩子户籍均在房产上
         → 最稳，基本保证入学

第二顺位：父母房产 + 孩子户籍在房产上
         → 也很稳，热门校可能卡这个

第三顺位：四老房产 + 父母无房 + 孩子户籍在此
         → 需要提供父母无房证明

第四顺位：父母房产 + 孩子户籍不在（跨区）
         → 有风险，热门校可能录不到这里

第五顺位：集体户/公共户
         → 需配合实际居住证明

第六顺位：非京籍有房
         → 由教委统筹派位

第七顺位：非京籍无房
         → 全区统筹，不确定性最高
```

## 三、各区顺位差异

### 海淀区

海淀的顺位执行最严格，特别是在热点学校（中关村三小、人大附小等）：

- 第一、二顺位基本能进（但多校划片下也要参加派位）
- 第三顺位（四老房产）被调剂的风险明显增加
- 1911后购房全部多校划片，顺位优势被削弱

### 西城区

西城的顺位是全市最复杂的：

- 对"户籍+房产"的一致性要求最高
- 四老房产顺位比海淀更靠后
- 租房家庭基本没有进入热点校的可能
- 731后购房统一多校划片

### 东城区

东城的顺位相对清晰友好：

- 四老房产的认可度较高
- 顺位之间的差距相对较小
- 即使顺位靠后，东城整体教育均衡，调剂结果也不会太差

### 朝阳区

朝阳的顺位在核心城区中最宽松：

- 有房非京籍的顺位待遇较好
- 热点校的顺位差距不那么悬殊

## 四、顺位对购房决策的影响

### 第一顺位最稳，但不是100%

即使你是第一顺位，在热点学校也可能面临：
- 学位不够 → 被调剂到片区内其他学校
- 多校划片 → 参加派位，结果不确定

### 四老房产的顺位在下降

越来越多区在缩紧四老房产的顺位待遇。部分区要求父母出具无房证明，有些区甚至要求父母和孩子户籍都在四老房产上且实际居住。

### 非京籍的顺位

非京籍是不参加顺位排队的——而是由区教委**单独统筹**。在非京籍群体内部，有房优于无房。

## 五、如何提升自己的顺位

### 最有效的方法

1. **父母房产 + 父母户籍落在房产上**（第一顺位）
2. **尽早落户**（部分区对落户年限有要求）
3. **保持户籍+房产一致**

### 需要注意的坑

- 父母有房但户籍在四老房产上 → 可能按四老房产算，顺位反而更低
- 夫妻双方只有一方户籍在房产上 → 可能算第二顺位
- 房产是夫妻一方婚前买的 → 需要确认户口迁入情况

## 常见问题

**Q：第一顺位保证进热点校吗？**
A：不保证。即使第一顺位，在多校划片政策下也要参加派位，且热门校学位有限。第一顺位不等于"锁定学位"。

**Q：租房能算第几顺位？**
A：租房不参加顺位排队。租房家庭的入学方式是通过区教委统筹安排，不在顺位体系内。

**Q：四老房产现在还能用吗？**
A：能用，但要求越来越严。多数区要求父母出具无房证明，且孩子和父母的户籍必须在四老房产上。建议提前咨询目标区教委确认具体政策。
