## 一、初三全年时间线

### 9月-12月（新课学习+初步复习）

**重点任务：**
- 完成初三新课程的学习
- 跟上老师的教学节奏，不留知识盲点
- 建立各科的错题本

### 1月-2月（寒假——第一轮复习）

**重点任务：**
- 系统梳理初中三年的知识体系
- 语文：背诵篇目全部过关
- 数学：基础概念+公式全面复习
- 英语：词汇+语法系统过一遍
- 物理：力学和电学重点突破

### 3月-4月（第二轮复习——专题突破）

**重点任务：**
- 各科专题训练（按题型分类）
- 开始做模拟试题
- 查漏补缺，针对薄弱知识点强化

### 5月（第三轮复习——模拟冲刺）

**重点任务：**
- 每周做1-2套完整模拟题
- 限时训练，适应考试节奏
- 调整作息，保持状态

### 6月（考前调整）

**重点任务：**
- 回归基础，不再追求难题
- 调整心态，保证睡眠
- 准备考试用品

## 二、各科复习建议

### 语文

- 古诗文默写：每天10分钟，反复记忆
- 现代文阅读：每周精读3-4篇，总结答题模板
- 作文：准备5个万能素材，灵活运用
- 字迹工整：语文考试卷面分很重要

### 数学

- 基础题（1-25题）：必须全部拿分，反复练习
- 压轴题（26-28题）：每天练1道，积累思路
- 错题本：每周回顾一次错题

### 英语

- 词汇：中考词汇每天背20个，循环记忆
- 听力：每天听15分钟，保持语感
- 作文：背10篇范文，掌握100个高级词汇

### 物理

- 力学和电学是重点，占分70%以上
- 实验题要理解原理，不是背步骤
- 公式要理解推导过程，不是死记硬背

## 三、每天的时间安排建议

| 时间段 | 做什么 |
|-------|-------|
| 6:30-7:00 | 起床、早读（语文/英语） |
| 7:00-17:00 | 学校上课 |
| 17:00-18:00 | 回家、休息 |
| 18:00-19:00 | 完成学校作业 |
| 19:00-20:00 | 专项复习（每天一科） |
| 20:00-20:30 | 英语/语文积累 |
| 20:30-21:00 | 回顾错题 |
| 21:00-21:30 | 预习明天的内容 |
| 21:30 | 休息 |

## 四、给家长的陪伴建议

1. **不要过度施压**——初三孩子的压力已经够大
2. **关注情绪变化**——如果孩子情绪低落，及时沟通
3. **后勤保障要做好**——营养、睡眠、运动都很重要
4. **模拟考成绩波动是正常的**——不要因为一次成绩下降就过度紧张
